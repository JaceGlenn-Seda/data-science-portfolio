# WHO/UNICEF Water Access Analysis — Global 2020

## Project Overview
A comprehensive data analysis of global drinking water access using the WHO/UNICEF JMP dataset for 2020, covering 210 countries across 4 water service levels: basic, limited, unimproved, and surface water.

## Tools Used
- Google Sheets — data cleaning, statistics, charts, pivot tables
- Python — independent validation of calculations

## Key Findings
- High-income countries average 99.55% basic water access vs 62.82% for low-income countries
- Rural areas have the highest variability (StdDev = 23.35) in basic water access globally
- Wealth and urbanization are the strongest predictors of water access
- South Sudan has the highest limited rural water access (42.16%) in the dataset
- Countries with 0% limited rural access average 99.29% national basic access
